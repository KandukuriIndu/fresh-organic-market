## ğŸ”§ Customization

### Change Brand Name
Find and replace "FreshMart" with your business name in HTML files.

Edit `css/style.css`:
```css
/* Primary color */
.a1 { color: #ff8c00; }
```

SVG files are editable text files. You can:
- Change colors by editing `fill` attributes
- Resize without quality loss
- Edit in any text editor or vector graphics software
